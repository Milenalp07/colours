## Overview
This project generates a random hex colour each time the user clicks a button.

The generated colour is applied to the background of a section element, and the hex code is displayed on screen.

## Features
- Generates a random hex colour (#RRGGBB)
- Changes the background colour dynamically
- Displays the current hex code
- Uses JavaScript event listeners
- Uses Font Awesome icon
- Styled with CSS and centred layout

## Technologies Used
- HTML
- CSS
- JavaScript
- Font Awesome
- Git & GitHub
- Vercel for deployment

## How It Works
1. The user clicks the button.
2. JavaScript generates a random hex colour.
3. The colour is applied to the section background.
4. The hex code is displayed on screen.
